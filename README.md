# نظام إدارة وجبات الكلاب (Dog Meals Manager)

تطبيق متكامل مبني بـ Flutter لمساعدة أصحاب مشاريع الأغذية والوجبات الطبيعية للكلاب على حساب التكاليف بدقة وإدارة العملاء.

## الميزات الرئيسية (Features)
- **شاشة الأسعار:** تحديث مباشر لأسعار الخامات (لحوم، دجاج، كبدة، بطاطس) ومصاريف التغليف والتشغيل.
- **حاسبة التكلفة الذكية:** 
  - تحسب التكلفة النهائية للوجبة بناءً على نسب الخلط الدقيقة.
  - تعوض تلقائياً نسب الفقد (Shrinkage) في التسوية (الدجاج والكبدة).
  - آلة حاسبة لفرق الدليفري (الخسارة التشغيلية).
- **إدارة العملاء (CRM):** 
  - تسجيل العملاء ونوع الوجبة المعتاد وتحديد دورة الطلب بالأيام.
  - نظام تنبيه ذكي يوضح العملاء الذين اقترب موعد نفاذ طعامهم.

## طريقة التشغيل (Setup & Run)
1. تأكد من تثبيت بيئة Flutter على جهازك.
2. قم بتشغيل التطبيق عبر الأمر:
   `flutter run`
3. التطبيق جاهز للعمل على الهواتف الذكية (iOS و Android) والشاشات المكتبية (Desktop) والويب (Web).
